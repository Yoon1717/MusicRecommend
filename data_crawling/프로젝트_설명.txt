4. 모델 생성 (NLP 자연어 처리)
	Gensim
	- Word2Vec : 단어 임베딩 기술을 사용, 단어를 벡터로 표현하는 모델
		: 주변 문맥 단어를 기반으로 단어의 분산 표현을 학습
		: 주변 단어들이 유사한 문맥에서 나타날 때 해당 단어들은 유사한 벡터 표현을 가짐
	- FastText : subword(글자 레벨)정보를 고려하는 모델 
		: 희귀한 단어나 오타가 있는 단어에 대한 강력한 벡터 표현을 얻을 수 있음
		: n-그램(subword)은 단어나 텍스트를 구성하는 문자나 음절을 고려하여 분석하는 텍스트 처리 기법
	
5. 임베딩을 파일로 저장
	: 텍스트 데이터에 대한 단어 또는 문장의 임베딩(벡터표현)을 보존하고 나중에 재사용 하기 위한 작업
	- 다른 작업 또는 모델 학습에 재사용이 가능하다 (시간과 계산 리소스를 절얄할 수있음)
